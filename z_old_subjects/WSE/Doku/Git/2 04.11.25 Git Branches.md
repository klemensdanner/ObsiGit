# Branches
```bash
git branch
```

Der 'Head' zeigt immer auf den aktiven Arbeitsbranch.

```bash
git branch alpha //erzeugt neuen branch, der auch auf die aktuelle Version zeigt
```

Jetzt können wir den 'Head' auf alpha zeigen lassen mit
```bash
git switch alpha
```

## git merge

```bash
git merge [Quellbranch]

//Ziel ist der aktuelle Arbeitsbranch
```

funktioniert gut, wenn unterschiedliche Bereiche geändert wurden.
Wenn in beiden branches die selbe Zeile editiert wurde - problem

```bash
$ git switch -c beta
```
