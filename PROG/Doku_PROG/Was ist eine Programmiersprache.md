
## Sprache:

Sprache ist die Menge aller Sätze.

Ein Satz hat eine Grammatik.

Grammatik:
- Satz (Syntax) - richtige Aneinanderreihung von Wörtern
- lexikalische Struktur - "sinnvolle Wörter": Bsp.; x?3:; ist kein sinnvolles Wort in unserer Sprache.


| Symbole mit bes. Bedeutung    | Verbundsymbole     | Schlüsselwörter (nicht als Variablennamen nehmen) |
| ----------------------------- | ------------------ | ------------------------------------------------- |
| + - * / = < > . , ( ) : ; ... | <> <= >= ... (* *) | AND ARRAY ASM BEGIN ELSE END                      |

### Pascal: Lexikalische Struktur
EBNF - Sprache zur Beschreibung von Syntax

letter: Terminal und Non-Terminalsymbole.

|        | Beschreibung | Beispiel                                   |
| ------ | ------------ | ------------------------------------------ |
| letter |              | 'A' \| 'B' \| 'C' ...<br>'a' \| 'b' \| ... |
| ident  |              | {letter \| digit \| '_'}                   |
